TeluguScript Documentation

TeluguScript is a programming language designed to honor Telugu Talli and celebrate the beauty of the Telugu language in the programming world. It is a transpiler that converts Telugu syntax into JavaScript, making coding an expressive and meaningful experience for Telugu-speaking developers.

With TeluguScript, developers can write code in Telugu, using native Telugu words as keywords and syntax. This allows for a deeper connection to the language while coding, making the development process more inclusive and culturally rich.

See official documentation for more details.

Installation

Install TeluguScript using npm:

npm install teluguscript

Features

Expressive Programming

Write code using Telugu keywords and syntax

Easy to Learn

Simple and intuitive syntax for native Telugu speakers

JavaScript Transpiler

Converts TeluguScript code into JavaScript for execution

Procedural Programming

Currently supports procedural programming paradigms

NPM Installable

Easy installation via npm package manager

Language Keywords

Variable Declarations

TeluguScript JavaScript Meaning
maredhi let Declare a mutable variable
maranadhi const Declare an immutable variable

maredhi - let

Use maredhi to declare a variable that can be reassigned.

maredhi name = "Siddhartha";
cheppu("Peru: " + name);
// Output: Peru: Siddhartha

name = "Vamsi";
cheppu("Updated Name: " + name);
// Output: Updated Name: Vamsi

maranadhi - const

Use maranadhi to declare a constant variable that cannot be reassigned.

maranadhi age = 21;
cheppu("Vayasu: " + age);
// Output: Vayasu: 21

// age = 22; // Error: Cannot reassign a constant variable

Control Flow

TeluguScript JavaScript Meaning
edi itey if Conditional statement
lekapote else Alternative condition
oka vela else if Additional condition

edi itey - if

Use edi itey to check conditions.

maredhi age = 25;
edi itey (age >= 18) {
    cheppu("Adult");
} lekapote {
    cheppu("Not an Adult");
}
// Output: Adult

oka vela - else if

Use oka vela when you want to check multiple conditions.

maredhi age = 20;
edi itey (age < 18) {
    cheppu("Underage");
} oka vela (age >= 18 && age <= 25) {
    cheppu("Young Adult");
} lekapote {
    cheppu("Adult");
}
// Output: Young Adult

Loops

TeluguScript JavaScript Meaning
malle itey while Loop while condition is true
malle for Iteration loop

malle itey - while

Use malle itey to loop as long as the condition is true.

maredhi counter = 0;
malle itey (counter < 5) {
    cheppu("Counter: " + counter);
    counter++;
}
// Output: Counter: 0
//         Counter: 1
//         Counter: 2
//         Counter: 3
//         Counter: 4

malle - for

Use malle to loop a specified number of times.

maredhi i;
malle (i = 0; i < 5; i++) {
    cheppu("i: " + i);
}
// Output: i: 0
//         i: 1
//         i: 2
//         i: 3
//         i: 4

Functions

TeluguScript JavaScript Meaning
pani function Define a function
phalitam return Return a value

pani - function

Use pani to define a function.

pani greet(name) {
    cheppu("Hello, " + name);
}

greet("Siddhartha");
// Output: Hello, Siddhartha

phalitam - return

Use phalitam to return a value from a function.

pani add(a, b) {
    phalitam a + b;
}

maredhi result = add(5, 10);
cheppu("Result: " + result);
// Output: Result: 15

Logging

cheppu - console.log

Use cheppu to print messages to the console.

cheppu("Hello, World!");
// Output: Hello, World!

Object-Related Keywords

TeluguScript JavaScript Meaning
paddathi class Define a class
edi this Reference to current object
kotha new Create new instance

paddathi - class

Use paddathi to define a class.

paddathi Person {
    pani initialize(name, age) {
        this.name = name;
        this.age = age;
    }
    
    pani greet() {
        cheppu("Hello, my name is " + this.name + " and I am " + this.age + " years old.");
    }
}

maredhi person = new Person("Siddhartha", 21);
person.greet();
// Output: Hello, my name is Siddhartha and I am 21 years old.

Asynchronous Programming

TeluguScript JavaScript Meaning
pedda pani async Define async function
agu await Wait for async operation

pedda pani - async

Use pedda pani to define an asynchronous function.

pedda pani fetchData() {
    cheppu("Fetching data...");
    agu new Promise((resolve) => setTimeout(resolve, 2000));
    cheppu("Data fetched!");
}

fetchData();
// Output: Fetching data...
//         (wait 2 seconds)
//         Data fetched!

Complete Example

Here's a comprehensive example demonstrating the usage of all keywords:

paddathi Person {
    pani initialize(name, age) {
        edi.name = name;
        edi.age = age;
    }
    
    pani greet() {
        cheppu("Hello, my name is " + edi.name + " and I am " + edi.age + " years old.");
    }
}

pedda pani fetchPersonData() {
    maredhi personData = { name: "Siddhartha", age: 21 };
    agu new Promise((resolve) => setTimeout(resolve, 1000));
    cheppu("Fetched data: " + personData.name + ", " + personData.age);
    phalitam personData;
}

pani main() {
    cheppu("Fetching person data...");
    maredhi personInfo = agu fetchPersonData();
    cheppu("Data fetched!");
    maredhi person = new Person(personInfo.name, personInfo.age);
    person.greet();
}

main();

// Output: Fetching person data...
//         (wait 1 second)
//         Fetched data: Siddhartha, 21
//         Data fetched!
//         Hello, my name is Siddhartha and I am 21 years old.
